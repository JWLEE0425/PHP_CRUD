# PHP_CRUD

### 概要
* PHPに作った簡単な**CRUD(Create, Read, Update, Delete)掲示板**です。

### 開発言語
* 言語 <br>
PHP
* DB <br>
MariaDB
* Javascript/HTML/bootstrap
### 機能紹介
#### 会員登録
> 会員登録です。IDとnicknameを入力します。
> IDとnicknameには重複チェック機能があります。
